# 🎥 Deepfake Detector using rPPG

## 📌 Project Overview

This project detects whether a video is **REAL or DEEPFAKE** using a biological signal called **rPPG (Remote Photoplethysmography)**.

---

## 🚀 Key Idea

Real human faces have heartbeat signals due to blood flow.
Deepfake videos do not have consistent biological signals.

---

## 🧠 Tech Stack

* Python
* OpenCV
* NumPy, SciPy
* Scikit-learn
* Streamlit

---

## ⚙️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📊 Output

* REAL
* FAKE
* UNCERTAIN

---
